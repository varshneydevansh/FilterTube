Filtering Guide

Everything you need to know about FilterTube's powerful filtering options

Comprehensive Filtering Guide

FilterTube provides powerful options to customize your YouTube experience. This guide explains how each filtering method works.

How FilterTube Works

1. Initial Hide

Content is hidden instantly via CSS as the page loads

2. Filter Processing

Your keywords and channel filters are applied to all content

3. Content Restoration

Content that doesn't match your filters is revealed

4. Layout Preservation

Layout fixes are applied to maintain proper display

5. Continuous Monitoring

As you scroll, new content is filtered automatically

Keyword Filtering

Keyword filtering checks video titles, descriptions, and channel names for specified terms.

  • Enter words or phrases separated by commas
  • Not case-sensitive (e.g, "news" will match "News" or "NEWS")
  • Uses whole-word matching for precision (e.g., "cat" will only match "cat" but not "category")
  • Multi-word phrases use exact phrase matching (e.g., "reaction video" only matches that exact phrase)
  • Useful for filtering topics, genres, or specific content types
Examples:

reaction video, unboxing, trailer

This would filter videos containing the exact phrase "reaction video" or the whole words "unboxing" or "trailer" in their titles or descriptions.

Pro Tip: The whole-word matching system allows more precise filtering. For example, using "cat" will only filter videos about cats, not ones mentioning "category" or "concatenate".

Channel Filtering

Channel filtering allows you to block content from specific channels using three different methods:

  • Channel Name: Filters using partial matching (will match any channel containing that text)
  • @Handle: Filters using exact matching (only blocks the exact @handle specified)
  • Channel ID: Filters using exact matching of alphanumeric channel IDs
Examples:

Gaming Channel, @OfficialNewsNetwork, channel/UCtxdfwb9wfkoGocVUAJ-Bmg

This would filter:

Channel names containing "Gaming Channel" The exact @OfficialNewsNetwork handle The exact channel with ID UCtxdfwb9wfkoGocVUAJ-Bmg
Important: Channel names use partial matching, so "Gaming" would match "Gaming Channel" and "Best Gaming Videos". @Handles and channel/IDs use exact matching to prevent unexpected filtering.

Comment Filtering

FilterTube provides two options for handling comments on watch pages:

  • Hide all comments: Completely removes the comments section from all videos
  • Hide comments with filtered words: Applies your keyword filters to comments, hiding only those containing filtered terms
Comment filtering options can be found in the extension popup under the "Comment Filtering" section. These options are mutually exclusive - enabling one will disable the other.

Advanced Filtering Techniques

Fine-tune your filtering with these advanced techniques:

  • Combining Filters: Use both keyword and channel filters together for more precise filtering
  • Specific Keywords for Channels: Filter channels without blocking their entire content by using distinctive channel phrases
  • Content Type Filtering: Use keywords like "shorts", "live", or "premiere" to filter specific content formats
  • Temporary Filtering: Quickly enable/disable filters by saving different filter combinations
FilterTube remembers your filters between sessions, so you can set them once and enjoy a cleaner YouTube experience consistently.

Layout Preservation

FilterTube uses sophisticated techniques to maintain YouTube's intended layouts when filtering content:

  • Horizontal Layout Preservation: Maintains proper side-by-side layouts for mixed content
  • Grid Layout Restoration: Preserves grid layouts on channel pages
  • Flex Layout System: Uses CSS flex properties to ensure content remains properly sized and positioned
  • Dynamic Layout Fixing: Automatically detects and fixes layout issues caused by filtering
The layout preservation system runs automatically when content is filtered - you don't need to configure anything.

Troubleshooting

If you encounter issues with filtering, try these solutions:

  • Content Not Being Filtered: YouTube occasionally changes its page structure. Try reloading the page or restarting your browser.
  • Over-Filtering: If too much content is being filtered, your keywords might be too broad. Try using more specific terms.
  • Layout Issues: If you notice layout problems, reload the page. FilterTube's layout preservation system will re-apply.
  • Persistent Problems: Try disabling and re-enabling the extension, or check for updates.
FilterTube is actively maintained to adapt to YouTube's evolving design. If you find persistent issues, please report them on our GitHub repository.
